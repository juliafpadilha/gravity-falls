# Dipper vs Mabel Race

Jogo inspirado nos gêmeos de Gravity Falls :)

## 👥 Integrantes

* Julia Padilha
* Bruno Hinz

## ▶️ Como executar

1. Abrir o arquivo `index.html`
2. Clicar em iniciar (ou pressionar ENTER)
3. Jogar localmente com 2 jogadores no teclado

## 🕹️ Como jogar
* Player 1: movimenta com W, A, S, D
* Player 2: movimenta com as setas do teclado
* Objetivo: chegar primeiro ao objeto em cada fase

## 🏆 Regras
* O jogo possui 5 níveis
* Cada nível dá 100 pontos para ambos os jogadores
* Quem chegar primeiro ganha a rodada
* No final, vence quem tiver mais vitórias
