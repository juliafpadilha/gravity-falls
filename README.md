# 🌲 Gravity Falls: Escape 🌲

Um jogo de plataforma cooperativo (ou competitivo!) desenvolvido em **p5.js**. Controle Dipper e Mabel em uma jornada para escapar das armadilhas e enfrentar o temível Bill Cipher através de 5 níveis desafiadores.

## 🎮 Sobre o Jogo

Inspirado na mecânica de clássicos como *Fireboy and Watergirl*, o objetivo é levar os personagens até o portal dourado (objetivo) no final de cada nível. 

### 🚀 Funcionalidades
*   **Multiplayer Local:** Jogue com um amigo no mesmo teclado.
*   **Física de Plataforma:** Sistema de gravidade, pulos e detecção de colisão.
*   **Plataformas Móveis:** Níveis dinâmicos que exigem sincronia.
*   **Inimigos Inteligentes:** O triângulo Bill Cipher patrulha áreas específicas para impedir seu progresso.
*   **Sistema de Pontuação:** O jogo rastreia quantos níveis cada jogador liderou até o final.

## 🕹️ Como Jogar

1.  Acesse o projeto no [p5.js Web Editor](https://editor.p5js.org/) (ou abra o arquivo index.html utilizando o botão direito e clicando no Open With Live Server)
2.  Pressione **[ESPAÇO]** no menu inicial para começar.
3.  Chegue ao final dos 5 níveis para ver o placar final e a tela de vitória.

### Controles

| Jogador | Personagem | Movimentação | Pulo |
| :--- | :--- | :--- | :--- |
| **P1** | Mabel (Rosa) | `A` e `D` | `W` |
| **P2** | Dipper (Azul) | `Setas Esq/Dir` | `Seta Cima` |

*   **R:** Reinicia o jogo em caso de Game Over ou após a vitória final.

## 🛠️ Tecnologias Utilizadas

*   [p5.js](https://p5js.org/) - Biblioteca JavaScript para criação de gráficos e interações.
*   **Lógica de Orientação a Objetos:** Classes para Jogadores e Inimigos facilitando a escalabilidade.

## 🗺️ Estrutura de Níveis

1.  **Nível 1:** Introdução aos controles e saltos básicos.
2.  **Nível 2:** Primeiro encontro com Bill Cipher (cuidado com a plataforma central!).
3.  **Nível 3:** Introdução de plataformas móveis e saltos temporizados.
4.  **Nível 4:** Desafio de precisão com perigos estáticos e patrulha rápida.
5.  **Nível 5:** O grande desafio final com sincronia entre plataformas e inimigos.
